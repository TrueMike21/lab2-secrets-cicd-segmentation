# Lab 2 – Secrets Injection, CI/CD, and Segmentation Model Training

This repository contains the implementation for **Lab 2**, which extends Lab 1 by adding **secure secrets management**, a **CI/CD pipeline**, and **training a custom image segmentation model** for house segmentation using aerial imagery.

The project follows modern **ML and MLOps best practices**, using **Python 3.11**, **Docker**, and **GitHub Actions**, with no Jupyter notebooks.

---

## Objectives

- Implement secure **secrets injection** using environment variables and GitHub Secrets  
- Set up a **CI/CD pipeline** to automate testing, building, and Docker image publishing  
- Generate pixel‑level masks from aerial images (Week 7 code reuse)  
- Train a **segmentation model** (e.g., UNet or DeepLabV3+)  
- Evaluate performance using **IoU** and **Dice score**  
- Document results in a **4‑page technical report**
